# Files-first Baseline Pointer

This directory is intentionally kept as a compatibility pointer.

The immutable files-first baseline for 2026-04-17 lives in:

`../files_first_official_baseline_20260417/`

The previous copies in this directory were byte-for-byte duplicates of the
official baseline (`engine.py`, `files_first_support.py`, result JSON, and
report markdown). Keep a single source of truth to avoid static-scan noise and
accidental edits to two baseline copies.
